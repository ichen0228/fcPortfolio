Project #3 Experimental Clock

Description

This is a breathing exercise clock that pulse with second().

Design Process

I started by deviding the seconds by 20 to get the remainder as frame number, and animated the pulse with circle size.

Reflections

I had to change my project last second so this is not my proudest work, but if I were to have more time with this project, I will probably find a way for the pulse to go faster, and to change the color and opacity of the circles with each pulse.
